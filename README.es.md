## 🕹️ ¿Qué es esto?

**Presencia Enriquecida de GeForce NOW para Discord** te permite mostrar el **juego real que estás jugando en GeForce NOW** directamente en tu perfil de Discord. ¡Ya no más estados genéricos de "Jugando a GeForce NOW"! 

Ahora mostrará el **título real, arte del juego y estado personalizado** en tiempo real. Perfecto para streamers, gamers y cualquiera que disfrute mostrar lo que está jugando de forma precisa. 🎮💚

---

## 🧠 Cómo funciona (explicación simple)

La aplicación se ejecuta de manera eficiente y silenciosa en segundo plano y:
1. **Monitorea** la actividad de GeForce NOW.
2. **Compara** el juego iniciado con una robusta lista interna de juegos.
3. **Envía** los datos precisos de la Presencia Enriquecida a tu cliente de Discord.
4. **Mantiene** el estado sincronizado mientras el juego esté en ejecución.

*Es 100% automática — literalmente no necesitas configurar nada tras la instalación.*
